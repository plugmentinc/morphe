## タイムライン

### Phase 0：MORPHE_1（2026年4月初旬）
**初期版** - 教科書通りのCVR最適化LP
- 12セクション構造
- モーフキャラクター9体配置
- カラーパレット（黒+白+虹色）
- レスポンシブ対応

**評価：** 構造が「ありきたりすぎる」とユーザーから指摘
- 競合制作会社のLPと同じパターン
- 訪問者の記憶に残らない

---

### Phase 1：The Living LP - Selection Edition (MORPHE_3)
**革新的構造** - 業界に前例のない体験型LP

**追加した要素：**
- Selection HERO（「このページを閉じてください」反転LP）
- Day Storytelling（7日間の旅）
- 進化するモーフ（10ステージ）
- Live Dashboard（リアルタイム数値表示）
- Living Comparison（他社比較表）
- Interactive Pricing（料金シミュレーター）

**評価：** デプロイ後、ユーザーから「やりすぎ」「シンプルなMORPHE_2が一番好き」とのフィードバック

---

### Phase 2：MORPHE_2 + 進化モーフ（最終形 - 現在）

**判断：**
- MORPHE_2のシンプルさを採用
- 進化モーフだけ追加（ブランド体験要素として保持）
- 革新的要素は全削除

**理由：**
1. クライアント（札幌の中小企業）にとっては「ちゃんとしてる感」が大事
2. 「閉じてください」と言われると、本気のクライアントでも引いてしまう
3. アニメーション付きLPだけで十分な差別化
4. シンプル=ビジネス的に正解

---

### Phase 3：コーポレート/LP分離構造（2026年5月）

**ChatGPT提案を採用：**
```
[トップ = LP]
├ /about → 会社情報

├ /contact → 問い合わせ
```

**ChatGPT案そのままではなく、以下を改善：**
- LPの「ありきたり構造」は採用せず、MORPHE_2版を維持
- 進化モーフを全ページに常駐
- ナビゲーションを別ページ遷移型に統一

## 主要な意思決定の理由

### 1. Selection HEROを削除した理由
- 「閉じてください」というコピーが攻撃的すぎる
- 札幌の中小企業ターゲットには合わない
- ブランドの信頼感を損なうリスク

### 2. 進化モーフを残した理由
- 唯一の業界差別化要素
- スクロールで楽しめるブランド体験
- 「ロゴなしでも記憶に残る」を実現

### 3. 買い切り＋サブスク両方提供する理由
- NORTH TECH（競合）はサブスクのみ
- 「選べる」ことで顧客層が広がる
- キャッシュフロー優先の小規模事業者と、所有権重視の中小企業、両方をターゲット

### 4. コーポレートをミニマルにした理由
- LPで派手にしているので、コーポレートは静かに
- 信頼感の演出
- クライアントが調査時に確認する「ちゃんとした会社か」を担保

## 競合分析

### NORTH TECH（thenorthtech.jp）
- **担当者**: 22歳の宮崎裕也（一人運営）
- **拠点**: 北海道・帯広
- **ターゲット**: 北海道の小さな店舗（個人サロン、キッチンカー、小規模飲食）
- **訴求**: 「10年使うHP」「近くの人」
- **価格**:
  - 梅: 月¥14,800 + 初期¥39,800
  - 竹: 月¥19,800 + 初期¥49,800（メイン）
  - 松: 月¥39,800 + 初期¥99,800
- **強み**: 圧倒的な低価格、解約金なし、最低契約期間なし
- **弱み**: テンプレ系、デザイン品質、規模対応力

### MORPHEのポジショニング

```
[品質高]
        MORPHE ← 中価格・AI×デザイン・高品質
            ↑
NORTH TECH      大手制作会社
（安い・店舗特化） （高い・遅い）
            ↓
        テンプレ系
        （Wix・STUDIO）
[品質低]
[安い]    →    [高い]
```

## 過去のフィードバックから学んだこと

| ユーザーの反応 | 学び |
|---|---|
| 「ありきたりに感じる」 | 教科書的構造は差別化にならない |
| 「やりすぎ」 | 革新性も度を超えると逆効果 |
| 「シンプルが一番好き」 | クリーンな美しさが最強の差別化 |
| 「モーフは可愛い」 | ブランド体験要素は維持すべき |

---

## 今後の方針

1. **MORPHE_2のシンプルさを基本姿勢として維持**
2. **進化モーフは絶対に削除しない**
3. **革新的要素を追加する場合は、ユーザーに必ず確認**
4. **「ありきたりじゃない」と「やりすぎない」のバランスを保つ**
5. **クライアント目線（札幌の中小企業）を常に意識**
