## 価格設定の根拠

### 市場調査結果（2026年5月）

| 依頼先 | 価格相場 |

| フリーランス | 5〜30万円 |
| **NORTH TECH（札幌・競合）** | 月¥14,800〜¥39,800 + 初期¥39,800〜¥99,800 |

| 中堅制作会社 | 80〜150万円 |
| 大手制作会社 | 200万円以上 |

### MORPHEのポジション

**「30〜100万円ゾーン」=** AI活用で工数削減 → 価格に反映 → 中小企業の主要レンジ

```
[品質高]
        MORPHE ← 中価格・AI×デザイン
            ↑
NORTH TECH      大手制作会社
（安い・店舗特化） （高い・遅い）
[安い]    →    [高い]
```

## 価格戦略の3つの柱

### 柱1：両モデル提供（買い切り＋サブスク）
- NORTH TECHはサブスクのみ → 「**選べる**」を強みに
- 買い切り = 所有権重視の中小企業
- サブスク = キャッシュフロー優先の小規模事業者

### 柱2：NORTH TECHの1.3〜1.5倍の価格
- 過度に安売りしない
- 業界標準よりは安い
- AI活用で工数削減 = 中価格帯で高品質を実現

### 柱3：明確なターゲット差別化
- NORTH TECH = 個人店・小規模店舗
- **MORPHE = 中小企業・スタートアップ**（より大きな案件）

## 価格表示UIの仕様

```html
<div class="price-tabs">
  <button class="price-tab active" data-tab="onetime">買い切り型</button>
```

## 運用方針

### 価格交渉について
- **STANDARD（人気No.1）を基準に提案**
- ボリュームディスカウント検討可能（年間契約等）
- 業界・案件規模により柔軟に対応

### 値上げのタイミング
- 案件が回り始めて3ヶ月後に評価
- 顧客満足度が高ければ20%値上げ可能
- 値上げ前に既存顧客には事前通知

### サブスク解約ポリシー
- **最低契約期間：なし**
- **解約金：なし**
- 解約は1ヶ月前通知（NORTH TECH対抗）

## 想定収益モデル

### 月10件成約時の試算

#### 買い切りメイン
- STANDARD（¥498,000） × 10件 = **¥4,980,000/月**
- 平均値: 概ね¥3,500,000〜¥5,000,000/月

#### サブスクメイン
- STANDARD（月¥29,800 + 初期¥98,000） × 10件
  - 初月: ¥98,000 × 10 + ¥29,800 × 10 = **¥1,278,000**
  - 翌月以降: 累積¥29,800 × 累積件数
- 1年継続で年商: ~ ¥7,200,000

#### ハイブリッド推奨
- 買い切り 5件 + サブスク 5件
- 月収益: 安定した売上 + ストック収益
